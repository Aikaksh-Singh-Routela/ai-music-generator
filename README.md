Generate simple melodies using AI-powered sound synthesis. Choose scales, adjust tempo, add effects, and download your creations as WAV files.

## 🚀 Features

- 🎵 **Multiple Scales**: C Major, G Major, A Minor, E Minor, Pentatonic
- 🎚️ **Adjustable Parameters**: Tempo, duration, number of notes
- 🔊 **Audio Effects**: Reverb and echo
- 📥 **Download WAV**: Save your generated melodies
- 📊 **Waveform Visualization**: See your music

## 🛠️ Tech Stack

- **Frontend:** Streamlit
- **Audio Processing:** NumPy, SciPy
- **Visualization:** Matplotlib

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/Aikaksh-Singh-Routela/ai-music-generator.git
cd ai-music-generator

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
🎮 How to Use
Adjust parameters in the sidebar (scale, tempo, duration)

Click "Generate Music"

Listen to your melody

Download as WAV or visualize the waveform
